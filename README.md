# Palindrome Birthday

### Description

###### (levelZero_markThirteen)

The app checks if your birthday is a palindrome or not. It checks 6 date formats - ddmmyyyy, mmddyyyy, yyyymmdd, ddmmyy, mmddyy, yymmd
